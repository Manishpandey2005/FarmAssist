🌾 FarmAssist Backend

✅ Server is running on port 8000

Available API Endpoints:

  • POST /auth/signup - User registration
  • POST /auth/login - User login
  • POST /predict/leaf - Disease detection
  • POST /predict/crop - Fertilizer recommendation
  • GET /weather - Weather data
  • GET /market - Market prices
  • GET /alerts - Pest alerts
  • POST /alerts - Submit pest alert

To run the full application:

  1. Start this backend server (already running)
  2. In another terminal: cd frontend && npm start
  3. Access frontend at: http://localhost:3000
